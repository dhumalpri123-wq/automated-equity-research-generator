Automated Equity Research Report Generator

A Python based Equity Research Tool that automatically generates professional PDF reports for NSE listed companies using financial data, valuation models, and automated report generation.

Features

- Dynamic NSE Ticker Input
- DCF Valuation Model
- Peter Lynch Valuation
- Relative Valuation Analysis
- Peer Comparison
- Financial Health Scorecard
- SWOT Analysis
- Investment Thesis
- Trend Analysis
- Automated PDF Report Generation

Tech Stack

- Python
- Pandas
- yFinance
- Matplotlib
- ReportLab

Sample Output

The tool automatically generates a professional equity research report containing:

- Executive Summary
- Valuation Summary
- Investment Recommendation
- Financial Performance Analysis
- Peer Comparison
- Financial Health Scorecard
- Trend Analysis
- Revenue Trend Charts
- Net Income Trend Charts
- Stock Price Charts
